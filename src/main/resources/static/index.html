How to prepare the Excel file
1
Create an Excel file (.xlsx or .xls) with these exact column headers in Row 1:
Email Shift Start Shift End From Date To Date
Column order does not matter. Headers are case-insensitive.
2
Add one row per person per shift block (date range):
Email Shift Start Shift End From Date To Date
alice@company.com 07:0016:00 01/03/202631/03/2026
bob@company.com 15:3000:30 01/03/202615/03/2026
bob@company.com 07:0016:00 16/03/202631/03/2026
charlie@company.com 22:3007:30 01/03/202631/03/2026
alice@company.com 15:3000:30 01/03/202631/03/2026
3
Important rules:
Each row is automatically expanded into one database entry per day within the date range — no need to list every date manually.
Multiple people on the same date — add multiple rows with the same or overlapping date ranges.
Person changes shift mid-month — add two rows for the same email with different date ranges and timings (see Bob above).
Overnight shifts are supported — e.g. Shift Start 22:30, Shift End 07:30 (next day).
Date formats accepted: dd/MM/yyyy  ·  MM/dd/yyyy  ·  yyyy-MM-dd  ·  or native Excel date cells.
Time formats accepted: HH:mm  ·  H:mm  ·  HH:mm:ss  ·  or native Excel time cells.
Uploading a new file for the same month replaces all existing entries for that month automatically.
The uploaded file is never saved to disk — it is read in memory and discarded immediately after processing.
